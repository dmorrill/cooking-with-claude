## Hacker News Version

**Title:** Show HN: Natural Language Cooking via Claude Desktop MCP

I built an MCP server that lets you manage cooking/recipes through conversation with Claude Desktop.

**Technical details:**
- MCP (Model Context Protocol) server in Node.js
- Reads/writes markdown files for data persistence
- 9 tools exposed: recipe search, inventory CRUD, meal planning
- Auto-detects repo location, zero config

**Why I built this:**
Every cooking app dies at data maintenance. You're tired after cooking, updating the app feels like work, system goes stale. Natural language finally fixes this.

**Interesting challenges solved:**
- Parsing markdown tables while preserving formatting
- Fuzzy ingredient matching across recipes/inventory
- Expiration date tracking without a database
- Meal suggestions ranked by available ingredients + expiration priority
