## Customization

This structure uses upstairs/downstairs/basement organization. Customize to match your home:

**Alternative structures:**
- Kitchen / Garage / Basement
- Main Floor / Second Floor
- House / Outdoor Kitchen
- Single location (just use one folder)

To customize:
1. Rename or delete folders in `inventory/`
2. Update this dashboard with new locations
3. Update `CLAUDE.md` with new file paths for inventory checks
